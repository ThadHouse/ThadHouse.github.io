The following is a list of many of the larger projects I have worked on and contributed to. For a more comprehensive list, see my GitHub Profile, the organization for my C# robotics libraries robotdotnet, and the wpilibsuite for the open source robotics project I contribute to.

  • vscode-wpilib A VS Code extension built to allow high school robotics teams to create, write and upload code to their robots. Works with multiple languages (C++ and Java) by default, and is extensible.

  • allwpilib The primary WPILib repo. I own and focus on the HAL subsystem, along with maintaining the build setup. In 2017, I rewrote and reimagined the HAL API to make it cleaner, easier to use, and more performant. This was my first large contribution to an external Open Source project. For the build system portion we currently use Gradle and CMake, with multiple custom written Gradle plugins in order to make multi platform builds possible.

  • C# FRC Utilities A collection of tools designed to make C# programming easier, especially related to interop. Contains tools for detecting native libraries, and generating classes at runtime to interface with those native libraries in the most performant way possible.

  • C# FRC CLI A command line utility for deploying robotdotnet code to the control system we use. This was an attempt to learn a bit more about unit testing, dependency injection, and mocking
